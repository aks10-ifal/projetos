GRID LAYOUT

Usando Grid Layout, crie o layout da imagem abaixo. Você deve usar cor de fundo e cor da borda para diferenciar as células. Não esqueça os espaçamentos (gaps) entre algumas células.

Noticias

Crie um site de notícias com o layout abaixo. O site deve ter uma notícia fixa dentro de cada grid item. A notíci deve ter um título e uma pequena descrição de até 100 palavras. Ao passar o mouse por cima do grid item um efeito deve acontecer para chamar a atenção do usuário. Assim como na imagem, o grid deve ser responsivo para aparecer corretamente no celular. As cores, a fonte, o efeito e as notícias ficam ao seu critério.

(Hu)go Template Primer

Hugo uses the excellent go html/template library for its template engine. Itis an extremely lightweight engine that provides a very small amount of logic. In our experience that it is just the right amount of logic to be able to create a good static website. If you have used other template systems from different languages or frameworks you will find a lot of similarities in go templates.
